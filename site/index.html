Adobe After Effects Script

Bulk Renamer

Rename every selected item in your Project panel in one pass. Find & replace text, add a prefix or suffix, or version up multiple comps at once.

Download Script (.jsx) View GitHub Repository

Version 1.0.3 · Single .jsx file · Dockable ScriptUI Panel

What it does

Four Ways to Rename

01

Find & Replace

Swap any text across all selected comp names at once

02

Prefix

Add text to the beginning of all selected comps' names

03

Suffix

Add text to the end of every selected comp's name — like "_textless"

04

Version Up

Version up every selected comp to its next free version number (v001 → v002), skipping numbers already taken elsewhere in the project.

How to use it

Installation

  1. Download the .jsx file using the button above.
  2. Open After Effects, then go to File → Scripts → Install ScriptUI Panel...
  3. Select the downloaded Bulk Comp Renamer v1.0.3.jsx file and confirm.
  4. After Effects will ask to restart — go ahead and restart it.
  5. Find it under Window → Bulk Comp Renamer.jsx.
  6. Select one or more items in the Project panel, fill in a field, and click the matching button.
